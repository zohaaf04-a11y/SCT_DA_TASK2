##  Data Cleaning & Exploratory Data Analysis using Python

### Objective

The objective of this task was to clean and prepare the Sample Superstore dataset using Python and Pandas in Google Colab.

### Tools Used

* Python
* Pandas
* Google Colab

### Dataset

Sample Superstore Dataset

### Files Included

* Sample_Superstore.csv (Raw Dataset)
* Cleaned_Superstore.csv (Cleaned Dataset)
* Data_Cleaning.ipynb
* Python_Data_Cleaning_Report.pdf
* Screenshots Folder

### Data Cleaning Steps

1. Loaded the dataset.
2. Checked for missing values.
3. Removed duplicate records.
4. Verified data quality.
5. Exported the cleaned dataset.

### Outcome

Successfully cleaned and prepared the dataset for further analysis.
